## Setup
- Add `nuxt-resources-loader` dependency using yarn or npm to your project
- Add `nuxt-resources-loader` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxt-resources-loader',

    // With options
    ['nuxt-resources-loader', { /* module options */ }],
 ]
}
```
